syn_probe attribute

syn_probe attribute works as a debugging aid. This attribute inserts probe points for testing and debugging the internal signals of a design, causing them to appear as ports at the top level.

Values:

1: Probe port names are generated by the synthesis tool.

value_pt: Probe port names ares identical to the syn_probe "value". if the signal is bus, the port name is adjusted to value_number

value_pt[]: Probe port names are identical to the syn_probe "value". if the signal is bus, the port name is assigned to value[number]
