syn_multstyle Attribute

The syn_multstyle attribute determines how multipliers are implemented, either as dedicated hardware multiplier blocks or as logic. You can apply it globally, to modules, or to multiplier instances of Altera or Xilinx devices.

ValueDescription
logic Maps to logic.
block_mult Maps to dedicated hardware multiplier blocks.

For detailed information about this attribute, use the Help button in the attribute wizard or the online help in the synthesis tool.
