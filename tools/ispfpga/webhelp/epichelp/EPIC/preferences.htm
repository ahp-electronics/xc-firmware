Setting Preferences

From within EPIC, you can add and delete certain preferences in the physical preference file (.prf). You edit the preference file whenever you make preference changes in EPIC and perform the Save command.

The Attributes dialog boxes enable you to set or change certain preferences. For example, when you change attributes in an Attributes dialog box or when you add and delete certain object types like paths and buses, you are writing out new preferences to the preference file.

The Tcl command line in EPIC allows you to set many other preferences. However, not all preferences are supported by EPIC.
