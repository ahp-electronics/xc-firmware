Opening Designs or Macros

After you have opened the EPIC software, you can open an existing design or existing macro.

To open an existing design:
1. Choose File > Open > Open Design. Alternatively, select a recently opened design file from the File > Recent Files menu.
The Open Design dialog box appears. If you have previously used EPIC, the dialog box will display the name and location of the .ncd file and .prf files of the last project.
To open this design, simply click OK.
To open a different design, use the Browse button to navigate to directory of the .ncd file.

If Auto lookup is selected and the design directory contains a preference file with the same root name as the .ncd file, EPIC will automatically open that preference file.

If Auto lookup is not selected, click the Browse button and select the name of the .prf file.
2. Click OK.
EPIC loads the design and displays the layout in the editing area of the main view.
To open an existing macro:
1. Choose File > Open > Open Macro.
2. In the dialog box, navigate to the macro file (.nmc), select it, and click Open.
EPIC loads the macro file and displays the layout in the editing area of the main view.
