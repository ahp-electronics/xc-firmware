Moving a Macro in Design Mode

To move a macro with placed components to an unused group of sites, use the swap command. This is quicker than unplacing the macro and placing it at the new location.

To move a macro to an unused group of sites:
1. Select the macro you want to move.
2. Select a vacant site to which you want to move the macro's reference component. The reference component and the selected site must be the same type of block: I/O block and I/O block or logic block and logic block.
3. Choose Edit > Swap. The reference component moves to the unused site, and the other components in the macro move to maintain their positions relative to the reference component.

Each macro's external pins are routed after the macro is placed. Any macro nets that were not pre-routed are also automatically routed.
