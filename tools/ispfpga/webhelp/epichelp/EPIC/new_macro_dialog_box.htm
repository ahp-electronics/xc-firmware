New Macro Dialog Box

The New Macro dialog box allows you to create a macro design file (.nmc) and select the targeted device.

Feature
Description
Macro file Browse Button
Allows you to navigate to the desired physical macro (.nmc) file.
Architecture List
Select the targeted architecture.
Device List
Select the targeted device family within the architecture.
Package List
Select the package technology.
Performance List
Select the performance grade.
OK Button
Applies the selected options to the current design and closes the dialog box.
Cancel Button
Ignores any options you may have specified and closes the dialog box.
Help
Opens the online Help topic for this dialog box.
