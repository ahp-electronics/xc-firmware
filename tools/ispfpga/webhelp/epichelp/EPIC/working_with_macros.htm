Working with Macros in Macro Mode

Most editing operations that you would perform in macro mode are identical to those performed in design mode. However, a number of the operations are different, and some of them can only be performed in macro mode.

Operations that are unique to macro mode are available from the Design > Macro menu. The selections from this menu can only be used in macro mode.

Macro Mode Operation
Function
Add External Pin
Designates that a selected component pin is an external pin; that is, the pin can be connected to external nets after the macro is instantiated in a design file.
List External Pins
Lists all external pins specified in the macro library file. 
Set Reference Comp
Designates a selected macro component as the macro's reference component. If the macro has at least one pre-placed component, it will have a reference component. When you place, move, or copy a macro you select a site for the reference component, and all of the other components are then located relative to the reference component. If you do not designate a reference component, the component in the lower left corner will be selected as reference component when the macro library file is saved. 
Get Reference Comp
Displays the name of the reference component in the Tcl console.
