Panning in the Main View

There are several ways to move around the main view after zooming in:

Use the mouse wheel
Use the arrow keys
Use the keys “h,” “l,” “j,” “k”
Use the space bar
Move the view indicator box inside the World View
To pan using the mouse wheel:
1. Click once inside the main view.
2. Press and hold the mouse wheel and drag the layout in the desired direction.
To pan using the arrow keys:
1. Click once inside the main view.
2. Scroll left, right, up, or down using the appropriate arrow key.
To pan using the h, l, j, and k keys
1. Click once inside the main view.
2. Use the “h” and “l” keys to pan horizontally. Use the “j” and “k” keys to pan vertically.
To pan using the space bar:
1. Click once inside the main view
2. Press the space bar once or repeatedly to shift the display incrementally, or hold the space bar to rapidly pan.
To pan using the World View:
Click inside the view indicator box, which is the highlighted square inside the World View. Press and hold the left mouse button and drag the square in the desired direction.
