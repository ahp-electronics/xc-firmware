Net Attributes Dialog Box

The Net Attributes dialog box allows you to view and select options for a selected net. Click the attributes button on the Actions panel or choose Design > Attributes to open this dialog box.

Feature
Description
Net Name
Displays the name of the selected net.
Route Status
Indicates whether the net is unrouted, partially routed, or fully routed.
Number of Pins
Indicates the number of pins contained in the net.
Lock Status
Indicates whether the object is locked or unlocked. A locked net cannot be unrouted or deleted. Pins and components connected to the net cannot be unplaced, swapped or deleted.
Lock Net
Locks the net by applying a LOCK NET preference. The net cannot be unrouted or deleted.
Block
Applies a BLOCK NET preference to the net. The BLOCK NET preference prevents timing analysis on the selected net and on all paths through it.
Use Longline
Applies a USE LONGLINE preference. The USE LONGLINE preference can be applied to wires, buses, and nets. When applied to a bus, the USE LONGLINE preference tries to route each net in the bus along a long line.
Pin List
Displays the name and type of each net pin of placed components and the delay time from the net source to each pin.
OK Button
Applies the selected options and closes the dialog box.
Cancel Button
Ignores any options that were selected and closes the dialog box.
Apply Button
Applies selected options without closing the dialog box.
Help
Opens the online help topic for this dialog box.
