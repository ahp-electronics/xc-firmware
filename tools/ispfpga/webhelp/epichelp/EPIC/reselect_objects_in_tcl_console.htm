Re-selecting Objects in the Tcl Console

Each time you select an object in the editing area, it is recorded in the Tcl console with a hyperlink. This enables you to quickly return to the object and re-select it at any time in the current session.

To re-select an object:
In the Tcl console, scroll through the list of pre-selected objects and click the hyperlinked name of the one you want to re-select.
The object is immediately selected in the editing area.
