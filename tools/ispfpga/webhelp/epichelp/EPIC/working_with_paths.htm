Adding and Deleting Paths

You can define a path in EPIC for timing analysis and timing-driven operations. When you add a path in EPIC, a DEFINE PATH preference is applied to the design.

To add a path:
1. Select the nets and components that will be included in the path.
2. Choose Edit > Add Path.

Take note of the following when adding paths:

You can also add a path by selecting the two endpoints of the path (that is, source pin and destination pin) and then choosing the Edit > Add Path command. When multiple paths exist for the two endpoints, EPIC defines each path.
EPIC only enumerates the components and nets of the endpoints in the path attribute. The Timing Wizard however, will know that all intermediate components and nets are implied.
The DEFINE PATH preference you create when you add the path is written into the design's preference file the next time the design is saved.

When you delete a path, you remove the DEFINE PATH preference that defines the path. You also remove all preferences that directly depend on the DEFINE PATH preference; for example, MAXDELAY PATH.

To delete a path:
1. In the Object menu, select the path to delete.
2. Choose Edit > Delete or click the delete button on the Actions panel.
The path is deleted. The next time you save the design, the DEFINE PATH preference that originally defined the path will be commented out in the preference file.
Note
You cannot delete individual nets or components from a path definition.
