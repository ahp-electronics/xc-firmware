Components Attributes Dialog Box

After a component is selected, clicking the attribute button on the Actions panel or choosing Design > Attribute will open this dialog box, where you can modify attributes of both PFU and PIO components.

Feature
Description
Name
Displays the name of the selected component.
Logic Strings
Displays a read-only list of current logic strings for the selected component.
Locate
When selected, applies a LOCATE preference to the selected component. The LOCATE preference places a component at a specified site and locks the component to the site. For a macro, the LOCATE preference places the macro's reference component at a specified site, places all of the macro's pre-placed components (that is, all components that were placed in the macro's library file) in sites relative to the reference component, and locks all of these placed components at their sites.
Lock
When selected, applies a LOCK placement preference to the selected component. A locked component cannot be unplaced, unrouted, moved, swapped, or deleted. If a LOCK PLACEMENT preference currently applies to the design, the Lock option is already selected and you cannot unlock the component by clearing the selection. If a macro is locked, all macro components that were pre-placed (i.e., components that were placed in the macro's library file) are locked. The macro cannot be unplaced.
OK Button
Applies the selected options and closes the dialog box.
Cancel Button
Ignores any selected options and closes the dialog box.
Apply Button
Applies any selected options without closing the dialog box.
Help
Opens the online help topic for this dialog box.
