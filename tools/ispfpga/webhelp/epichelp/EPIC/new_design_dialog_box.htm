New Design Dialog Box

The New Design dialog box allows you to create a design file and its associated preference file, and select the targeted device.

Feature
Description
Design file Browse Button
Allows you to navigate to the desired physical design (.ncd) file. Alternatively, you can specify the full path to the file in the input box.
Preference file Browse Button
Allows you to navigate to the desired preference (.prf) file. Alternatively, you can specify the full path to the file in the input box.
Auto Lookup Option
When selected and no preference file is specified, EPIC assumes that the preference file has the same filename (but different extension) as the design file.
Architecture List
Select the targeted architecture.
Device List
Select the specific device family within the architecture.
Package List
Select the package technology.
Performance
Select the performance grade.
OK Button
Applies the given options to the current design and closes the dialog box.
Cancel Button
Ignores any options you may have specified and closes the dialog box.
Help Button
Opens the online Help topic for this dialog box.
