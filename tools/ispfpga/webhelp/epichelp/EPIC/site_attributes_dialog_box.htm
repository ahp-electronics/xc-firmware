Site Attributes Dialog Box

After a site is selected, clicking the attribute button on the Actions panel or choosing Design > Attribute will open this dialog box.

Feature
Description
Name
Displays the site name
Prohibit
If selected, applies a PROHIBIT SITE preference, which prevents the placement of any component in the site. You can use the PROHIBIT preference to reserve a site for future use.
OK Button
Applies the option and closes the dialog box.
Cancel Button
Ignores the option you may have specified and closes the dialog box.
Apply Button
Applies the given option without closing the dialog box.
Help
Opens the online help topic for this dialog box.
