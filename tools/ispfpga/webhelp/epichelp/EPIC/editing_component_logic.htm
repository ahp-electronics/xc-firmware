Editing Component Logic

The Logic Block view provides an interactive way to change component internal logic and configuration. When you open a Logic Block view, EPIC displays either a window with a schematic representation of the interior of the selected component (for components such as SLICE or PIO elements) or a tabular view (for EBR, PLL, DSP, SERDES). You can work within the Logic Block view to modify the component configuration.

To run the Logic Block view:
1. Select a component to edit in the editing area of the main view or in the Object menu.
2. Click the Zoom To button on the toolbar to enlarge and highlight the component.
3. Do one of the following:
Choose Edit > Edit Block
Click the editblock button on the Actions panel.
Double-click a component in the editing area of the main view.
The Logic Block view opens in a separate view and displays a logic view of the component—a schematic or tabular view. The view is a read-only dimmed view when you first open it.
4. To enable editing, click the Edit button on the Logic Block view toolbar.
5. If the view is a schematic, click the Attributes button on the toolbar to examine the attributes of the PIO or SLICE component.
6. After each design change, click the DRC button on the Logic Block view toolbar to run design rule check.
The DRC results are displayed in the Tcl console.
7. When you have finished editing the component, click the Stop Edit button.
The logic block view returns to a read-only view and is dimmed.
8. To return to the main view in EPIC, click the Back button.
When you return to the main view using the Back button, the logic block view remains open. You can return to it by selecting it from the Window > Opened Block Views menu.
