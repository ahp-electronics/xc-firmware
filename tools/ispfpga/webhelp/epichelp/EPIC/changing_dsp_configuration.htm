Changing DSP Configuration

DSP (digital signal processing) blocks are used to perform high-speed, fixed-point functions such as multiply, sum, and accumulate.

When you open the Logic Block view for a DSP element, a tabular view opens, which enables you to modify how the DSP is configured within your design, such as changing the module configuration or port interface.

See the FPGA Libraries Manual for details on attribute settings of individual DSP design elements. You can also refer to the following Technical Notes.

LatticeECP-DSP sysDSP Usage (TN1057)
LatticeECP2/M sysDSP Usage Guide (TN1107)
LatticeECP3-DSP sysDSP Usage Guide (TN1182)
LatticeXP2 sysDSP Usage Guide (TN1140)

You should do a DRC check after making changes. Click the DRC button on the Logic Block view toolbar. DRC results appear in the Tcl console.
