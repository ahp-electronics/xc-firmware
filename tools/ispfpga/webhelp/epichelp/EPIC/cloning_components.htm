Cloning Components

When you clone a component, you duplicate the logical contents of one PIC or PFU (the source) into another PIC or PFU (the target). Both components must be programmable. The source is a placed or unplaced component. The target can be a vacant site or a placed component. Only the programming of the component is copied; connections are not copied.

The target must be the same type of component as the source component. If the target is a site, it is programmed as a copy of the source. In the process, a new component is created. If the target is a placed component, its programming is overwritten by the source component's logic.

To clone a component:
1. Select the component that you want to clone. This is the source component.
2. While holding the Ctrl key, select the component or site into which you want to copy the source component's logic. This is the target component.
3. Choose Edit > Clone.
The logical contents of the source component are copied into the target component.
