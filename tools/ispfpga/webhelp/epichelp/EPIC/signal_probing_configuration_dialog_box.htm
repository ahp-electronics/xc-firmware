Signal Probing Configuration Dialog Box

This dialog box appears when you choose Design > Signal Probing.

Feature
Description
Signals List
List of available signals in your design that can be probed. These are paired with “receiver” grid cells from the Available IO list on the right.
The Wildcard filter allows you to filter signal names using a specific name or a wildcard value. For example, typing d* and pressing the Enter key will display all signals beginning with the letter “d.” To restore the list to its original state, enter * in the text box.
Available IO Column
These are available I/O grid cells that can be paired with the list of available signals in the design. You can click the column title to sort rows by IO.
Signal Column
This column’s cells are populated with the signals that you drag from the Signals list on the to pair them with the available I/O. You can click the column title to sort the rows by Signal.
Status Column
Shows the availability of the I/O or the routing status of a signal probe pairing. You can click the column title to sort rows by status.
Refresh Button
Refreshes the dialog box to display all of the changes you may have made to a signal in the main view, such as deleting a signal occupying an I/O site.
Dismiss Button
Closes the dialog box but does not cancel the signal probing changes you made.
Import Button
Imports a signal probing list. The list is a simple text file that lists the probed signals.
The Import and Export buttons allow you to save a signal probing list and then reuse it after the .ncd file is regenerated.
Export Button
Saves the current signal probing list as a simple text file. After the .ncd file is regenerated, you can reuse the same list by clicking the Import button. You can also manually edit the text file and import it again.
Help
Opens the online help topic for this dialog box.
