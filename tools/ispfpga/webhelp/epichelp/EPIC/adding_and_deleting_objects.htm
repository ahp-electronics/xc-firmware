Adding and Deleting Objects

You can add or delete objects by using the Actions panel or the Edit menu.

Adding Objects

When you add objects to a design, these new objects become part of the design database. You can add new components, nets, and macros; add component pins to existing nets; and add nets to existing buses. You can also define paths and buses.

To add an object to the database:
1. Select an appropriate vacant site or unplaced object in the design.
You will need to select more than one object to add or define certain objects; for example, two pins must be selected to define a path.
2. Choose the appropriate add command from the Edit menu or click the add button on the Actions panel.
An Add dialog box will appear for the type of selection you made.
3. Enter the required information and click OK.
Deleting Objects

Deleting an object permanently removes it from the current design. You can delete components, nets, net pins, macros, paths, and buses.

To delete an object:
Select the object, and then click the delete button on the Actions panel or choose Edit > Delete.
