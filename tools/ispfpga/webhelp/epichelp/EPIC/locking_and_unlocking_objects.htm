Locking and Unlocking Objects

You can lock objects to prevent objects from being deleted, unplaced (for components) or unrouted (for nets). This can be set in a preference file or by using the attributes dialog boxes in EPIC. If a component is locked, you cannot unplace it, but you can unroute it. To unplace the component, you must first unlock it. If a net is locked, you cannot unroute any portion of the net (that is, the entire net, a net segment, a pin, or a wire). To unroute the net, you must first unlock it. You can, however, add pins or add routing to a locked net.

To lock an object:
1. Select the object in the editing area of the main view.
2. Click the attribute button on the Actions panel or choose Design > Attribute.
If a LOCK preference already exists for the selected object, the Lock option in the Attributes dialog box will show a check mark.
3. In the Attributes dialog box, select the Lock option and click Apply.
4. Click OK to confirm and dismiss the Net Preferences dialog box.
To unlock an object:
Follow the same procedure for locking an object, but clear the Lock option.
