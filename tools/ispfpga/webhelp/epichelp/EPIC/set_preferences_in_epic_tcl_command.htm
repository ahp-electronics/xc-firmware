Setting Preferences in the EPIC Tcl Command Line

The Tcl command-line tool, located at the bottom of the EPIC Tcl console, enables you to set many preferences using the eco_config command. These preferences include attributes for IO, PLL, and memory.

When you enter a command, it is immediately recorded in the Tcl console and added to the .tcr file for the current session.

See the EPIC Tcl Commands section for details about setting preferences using eco_config.
