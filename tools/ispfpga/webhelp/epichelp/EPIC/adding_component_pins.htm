Adding Component Pins

Component pins can be added to nets if the component is placed in the design.

To add component pins to an existing net:
1. In the editing area of the main view, select the component pins that you want to add to a net. The selected pins must be on a placed component, and the selected pins must not belong to any other net.
2. Do one of the following:
In the Object menu, select the net to which you will add the pins.
In the editing area of the main view, select any net pin, ratsnest line, or routed segment on the desired net.
3. Click Add.

The new net pins are routed to the net.
