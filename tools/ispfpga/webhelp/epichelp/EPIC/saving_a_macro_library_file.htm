Saving a Macro Library File

At any time during an EPIC session, you can save the macro library file on which you are working.

When you save a macro library file, you cannot save a preference file along with the .nmc file, even if preferences were created when the macro library file was edited. All preferences are lost when you save the .nmc file.

To save your current macro library file:
Choose File > Save. The macro library file is written to disk.
To save a macro library file as a another file name:
1. Choose File > Save As. The Macro Save As dialog box appears.
2. In the Save As Macro Dialog Box, enter the macro filename under which you would like to save the design in the EPIC window, or use the Browse button.
3. Click OK.
The dialog box closes and the design is saved as a macro library file with the name you specified. The title bar changes to show the name of the new file. When you later save or exit the file, it will be saved to the new macro library file, not the original file.
Note
Macro files have an .nmc extension. If you do not add the extension .nmc, the extension automatically is appended to the file name. If you do not enter a full path name, the file is placed in the current working directory.
