Notes About Selecting

Buses, paths, and unplaced components and macros cannot be selected in the editing area of the main view. You can select these objects by using the Object menu. Select the object type from the drop-down menu, and then choose the desired net, component, macro, bus, or path.
When you select a component, net, or macro name from the Object menu or in the editing area, it becomes highlighted in the editing area. To make changes to an object, however, you must select it in the editing area.
When you click an object to select it, the previously selected item is no longer selected.
Multiple objects can be selected by using the Ctrl+Click combination.
When you click in the background, all previously selected items are no longer selected. You can also choose View > Unselect All to clear all selections.
When you click a net, the entire net becomes selected. To select a single wire, use the Alt+Click combination.
Switch box pins are selectable. When you select a switch box pin, the display will highlight the allowable connections from the selected pin to other pins within the switch box. You cannot perform any other operations on a selected switch box pin.
You can only display the connections for one switch box pin at a time. All other paths are cleared when a new switch box pin is selected within a single switch box.
When you select a ratsnest line, the entire net containing the ratsnest line is highlighted, and the net is selected.
