Changing I/O Types

To change I/O type, select a PIO component and open the Logic Block view. You will see a list of available I/O standards on the upper-left section of the view. Different architectures have different schematic layouts.

To change I/O types, select the option next to the desired I/O type. You can also select other options, when applicable, such as SLEWRATE, PCICLAMP, PULLMODE, DRIVE and VREF voltages.

For details about I/O usage, refer to the following Technical Notes:

LatticeECP2/M sysIO Usage Guide (TN1102)
LatticeECP3 sysIO Usage Guide (TN1177)
PURESPEED I/O Usage Guide (TN1088)
LatticeECP/EC and LatticeXP sysIO Usage Guide (TN1056)
MachXO sysIO Usage Guide (TN1091)
MachXO2 sysIO Usage Guide (TN1202)

You should do a DRC check after making changes. Click the DRC button on the Logic Block view toolbar. DRC results appear in the Tcl console.
