Add Macro Dialog Box

The Add Macro dialog box can be accessed by choosing Edit > Add Macro. It allows you to add a macro to your current design.

Feature
Description
Macro Name
Specifies a name for the macro instance. If you do not enter a name, a name will be assigned automatically when the macro is added to the design. The macro name appears in the attributes dialog box. If you change the macro name, all of the names of the components, nets, and external pins that are part of the macro will be updated automatically to reflect this name change.
Macro Library File
Allows you to specify a macro library filename. The associated Browse button allows you to navigate to an existing .nmc file.
OK Button
Applies the given options to the current design and closes the dialog box.
Cancel Button
Ignores any options you may have specified and closes the dialog box.
Help
Opens the online help topic for this dialog box.
