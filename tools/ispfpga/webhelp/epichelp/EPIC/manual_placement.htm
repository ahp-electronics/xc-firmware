Manual Placement

There are three manual placement procedures:

Manually placing an unplaced component or macro to a vacant site.
Moving a component or a macro to another site after it has been placed.
Swapping the sites of two placed components.
To manually place an unplaced component:
1. In the editing area of the main view, select the desired vacant site.
Note
The component and the site must be the same type of block (PIC and PIC, or PFU and PFU). Also note that Steps 2 and 3 and can be performed in reverse order.
2. In the Object menu, select Unplaced Comps from the drop-down list, and then select the component to be placed in the selected site.
3. Choose Design > Place > Manual Place.
The component is placed in the site and the component is routed after it is placed.
