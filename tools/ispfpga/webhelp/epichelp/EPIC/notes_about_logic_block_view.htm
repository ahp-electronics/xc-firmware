Notes About Using the Logic Block View

The Logic Block view always opens as a separate view. This allows you to return to the editing area of the main view, by clicking the Back button, while keeping the logic block view open in the background.
A logic block view will remain open until you click the close button in the top right corner.
You can have multiple logic block views open at once. To return to one of the opened logic block views, select it from the Window > Opened Block Views menu.
You can zoom in and out of a schematic logic block view using the following methods:
Drag to the right and downward with the mouse to zoom in, and drag upward and to the left to zoom out.
Use the keyboard combinations Ctrl++ to zoom in and Ctrl+- to zoom out.
While pressing the Ctrl key, move the mouse wheel forward to zoom in and backward to zoom out.
You can pan, after zooming in on a schematic view, by pressing the mouse wheel and dragging the layout in any direction.
As you click on a symbol in a schematic display, the name of the symbol appears in the Tcl console. This can be useful in understanding DRC warning and error messages.
Blue (cyan) lines in a schematic display indicate connections that have been programmed. Magenta lines indicate other available connections within the component.
