Logic Block View

After a component is selected, you can open the Logic Block view using one of the following methods:

Click the Editblock button on the Actions toolbar.
Choose Edit > Editblock.
Press Ctrl+E.
Double-click the component.

When you open a block in the Logic Block view, you see either a schematic representation of the interior of the selected component (for components such as SLICE or PIO), or a tabular view (for components such as EBR, PLL, DSP, SERDES). Click the Edit button on the Logic Block toolbar to make the view editable.

Feature
Description
Logic Block
Shows a schematic diagram or tabular view of all logic in the selected component.
Edit/Stop Edit
Toggles the Logic Block view between a dim uneditable view and a clear editable view.
Attributes Button
Opens the Components Attributes Dialog Box.
Equation Button
Opens the Equations Dialog Box for component logic editing.
DRC Button
Performs a Design Rule Check on the current configuration of the component, posting any errors and warnings to the EPIC Tcl console.
Restore Button
Resets the component configuration to the state that it was in when you opened the Logic Block view.
Back
Minimizes the Logic Block view and returns to the EPIC main window.

For general information on editing components, see the topic Editing Component Logic.
