Highlighting Objects

Objects in the editing area of the main view can be highlighted by using the View menu or the Actions panel.

By default, the objects are highlighted in yellow. You can change the highlight color in the Color tab of the Options dialog box. The objects in the same layer can only be highlighted in the same color. If the highlight color is changed, both the previously highlighted objects and the objects you highlight after that will show the new color.

To highlight an object:
1. Make sure that the layers displayed include the objects that you want to highlight. If not, select the desired layers from the toolbar.
2. Select the object that you want to highlight.
3. Click the hilite button on the Actions panel or choose the Highlight command from the View > Highlight/Unhighlight menu.
To change the highlight color:
1. Choose Edit > Option to open the Options dialog box, and then select the Color tab.
2. Click the color box for Highlight to open the Select Color dialog box.
3. Select a color from the Basic Colors or create a custom highlight color and click OK.
4. Click OK to apply the new color and close the Options dialog box.
To unhighlight highlighted objects:
Choose View > Highlight/Unhighlight > Unhighlight to unhighlight a select object.
Choose View > Highlight/Unhighlight > Unhighlight All to clear all highlighting.
