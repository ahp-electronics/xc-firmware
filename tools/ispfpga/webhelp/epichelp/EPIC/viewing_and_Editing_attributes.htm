Viewing and Changing Attributes

You can view and change a number of attributes or preferences for many EPIC objects. To open the attribute dialog box for a selected object, click the attributes button on the Actions panel or choose Design > Attributes.

The following table lists these objects and associated dialog boxes. Changes to some attributes will be written into the design's preference file when you save the design.

Attribute Dialog Box
Components
Component Name, Preferences and Logic Strings

External Pin Attributes Dialog Box

Macro Attributes Dialog Box
Buses
Bus Name, Elements, and other Preferences
Bus Attributes Dialog Box

Path Attributes Dialog Box
