Swapping Component Pins

The swap command enables you to swap pins on a component. EPIC only allows you to swap pins that are permitted for the mode in which the component is programmed. When the pins are swapped, the internal component logic is modified to match the new pin positions.

To swap component pins:
1. Select the two pins you wish to swap.
2. Choose Edit > Swap or click the swap button on the Actions panel.

The two pins are unrouted, and then they swap locations and are rerouted. The programming of the component changes to reflect the pin swap.
