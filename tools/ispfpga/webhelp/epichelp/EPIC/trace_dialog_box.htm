TRACE Dialog Box

The TRACE dialog box allows you to run timing analysis on the design. The results are displayed immediately in the dialog box. The TRACE dialog box opens when you choose Design > Trace Report.

Feature
Description
Report
Select the type of report: one that reports errors only or the verbose report.
Analysis type
Allows you to select the preferences for analysis in the “Available timing related preferences” section or use Default enumeration. Default enumeration runs a verbose timing analysis report without considering the effect of the constraints.
Number of items to report
Limits the number items reported to the maximum specified or reports an unlimited number.
Available timing related preferences
Displays a list of timing-related preferences in the design. If “Selected preferences” has been selected as the Analysis Type option, you can select the preferences you want analyzed from this list.
Run Button
Runs the timing analysis and displays the results in the “Available timing related preferences” section.
Cancel Button
Ignores any options you may have specified and closes the dialog box.
Help
Opens the online help topic for this dialog.
