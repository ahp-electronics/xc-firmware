EPIC Tcl Commands

The EPIC Tcl command line, located at the bottom of the Tcl console, enables you to use commands to perform the following basic actions. When you create Tcl commands, they are automatically saved in a .tcr file for the current EPIC session.

EPIC Tcl Commands 
Operation
Syntax
Description
New Design or Macro
eco_design new -arch <arch> -device <device> -package <package> -speed <speed> -ncd <ncd file> [-macro] [-prf <prf file>]
Creates a new design or macro and specifies the targeted device, performance grade, and .ncd file name.
Load Design or Macro
eco_design open -ncd <ncd file>
[-prf <prf file>] [-macro]
Opens an existing design or macro.
Save Design
eco_design save [-ncd <ncd file> -prf <prf file>] | [-nmc <nmc file>]
Saves the design or macro that is currently open or saves it in a different location or with a different .ncd file name.
Close Design
eco_design close
Closes the design or macro that is currently open.
Query Net/Comp/Site/Pin
eco_design query [-a]{ -net <net name> } | { -comp <comp name> } | { -pin <pin name> } | { -site <pin name> }
Searches for a specific net, component, site, or pin.
DRC
eco_design drc –all | -chip | { -net <net name> } | { -netsanity <net name> } |
{ -comp <comp name> } { [-err] | {[-unrouted] [-unplaced]} }
Performs design rule check on all objects or on selected nets or components. Options include chip check and net sanity check.
Add Component
eco_add comp [-name <comp name>]
-site <site name>
Adds a component to a specified site.
Add Net
eco_add net [-name <net name>] -netpin <pin name> [-netpin <pin name>] …
Adds a net that will connect two specified pins. The pins must be on placed components.
Add Pins to Existing Net
eco_add netpin -net <net name> -netpin <pin name> [-netpin <pin name>] …
Adds pins to an existing net.
Add Macro
eco_add macro -macro <macro name>
-nmc <nmc file> [-site <site name>]
Adds a macro to the design and specifies the location.
Delete Component
eco_delete comp <comp name> …
Removes a component from the design and unroutes each net pin on the component.
Delete Net
eco_delete net <net name> …
Removes a net from the design and unroutes and deletes each net pin from the design database.
Delete Net Pin
eco_delete netpin <pin name> …
Unroutes a signal pin and removes it from the net.
Delete Macro
eco_delete macro <macro name> …
Removes a macro from the design.
Swap or Move Component
eco_swap comp -comp1 <compname1>
{ -comp2 <compname2> } |
{ -site <sitename> }
Swaps comp1 with comp2 or moves comp1 to an empty site.
Move Macro
eco_swap macro -macro <macroname>
-site <sitename>
Moves a macro to an empty site.
Swap Pin
eco_swap pin <pinname1> <pinname2>
Swaps pin locations on a placed component.
Clone Component
eco_clone comp <compname1>
{ -comp <compname2>} |
{ -site <sitename> }…
Copies a component from one site and duplicates it in another site.
Clone Macro
eco_clone macro <macroname1>
{ -macro <macroname2>} |
{ -site <sitename> }…
Copies a macro from one site and duplicates it in another site.
Rename Component
eco_rename comp <old compname> <new compname>
Gives a component a new name.
Rename Net
eco_rename net <old netname> <new netname>
Gives a net a new name.
Auto Route
eco_route auto
{ -all
    [-i <iterations:-1,2000>]
    [-cc <costclean:0,20>]
    [-dc <delayclean:0,100>]
    [-ec <delaycleanif:0,100>]
    [-x] } |
{ -net <net name> } | { -comp <comp name> } | { -macro <macro name> |
{ -netpin <pin name> } | { {-netpin <pin name>} | {-wire <wire name>} {-netpin <pin name>} | {-wire <wire name>} } }
Auto routes all objects or specified nets, comps, or macro.
Additional options for -all flag:
-i    Performs the specified number of iterations.
-cc  Performs the specified number of cost-based cleanup iterations
-dc Performs the specified number of delay-based cleanup iterations
-ec  Performs the specified number of delay-based cleanup iterations if fully routed
-x   Disables timing-driven routing.
Manual Route
eco_route manual {[-netpin <pin name>] | [-wire <wire name>]} …
Routes the specified pins or wires in the order given.
Unroute
eco_route unroute [-all] | [-comp <comp name> ...] [-wire <wire name> ...] [-netpin <pin name> ...] [-net <net name> ...]
[-macro <macro name> ...]
Unroutes the connections specified. When the -all flag is used, all routed connections in the design are unrouted except locked signals and certain macro nets.
Auto Place
eco_place auto
[-x] [-l <level:1,5>] [-t <tableentry:1,100>]
Automatically places all components, with the following options:
-x  Disables timing-driven placement
-l  Specifies the placement effort level, with 1 being least difficult and 5 the most difficult.
-t  Specifies a cost table (from 1 to 100) for placement
Manual Place
eco_place manual
{ -comp <unplaced comp name> } |
{ -macro <unplaced macro name> }
-site <site name>
Places the specified unplaced components or macros into sites.
Unplace
eco_place unplace [-all] | [-comp <comp name> ...] [-macro <macro name> ...]
Unplaces all components or macros or only the ones specified.
Set Program String of Component
eco_config comp –comp <comp name> <logic string>
Sets the program string of the specified component.
Get Program String in Component
eco_config comp –comp <comp name>
Displays the program string of the specified component.
Configure sysIO
eco_config sysio -comp <comp name>
[-bank_vccio <bank_vccio>] [-clamp <clamp>] [-diffcurrent <diffcurrent>]
[-diffdrive <diffdrive>] [-diffresistor <diffresistor>] [-drive <drive>]
[-eq_call <eq_call>] [-hysteresis <hysteresis>] [-impedance <impedance>] [-io_type <io_type>] [-multdrive <multdrive>] [-opendrain <opendrain>]
[-pciclamp <pciclamp>]
[-pullmode <pullmode>] [-pwrsave <pwrsave>] [-refcircuit <refcircuit>]
[-slewrate <slewrate>] [-terminategnd <terminategnd>] [-terminatevccio <terminatevccio>]
[-terminatevtt <terminatevtt>]
[-termination <termination>] [-vcmt <vcmt>] [-vref <vref>]
Configures the I/O attributes for the specified component.
Configure PLL
eco_config pll -comp <comp name>
-type <type> [-phaseadj <phaseadj>]
[-delay_val <delay>] [-fdel <fdel>]
[-clkos_fdel <clkos_fdel>]
Configures the attributes for the specified PLL module.
Configure UFM
eco_config memufm -comp <comp name> -init_all_0 <no|yes> [-format <hex|bin>] [-mem <mem file>]
Configures the settings for the specified user flash memory component.
Configure EBR
eco_config memebr -instance <instance name> -init_all <0|1|no> -mem <file name> -format <hex|bin|add> -init_data <dynamic|static> -module <module name> -mode <mode value> -depth <depth value> -widtha <widthA value>
-widthb <widthB value> [-area <area value>] [-ecc <ecc value>] [-bytesize <bytesize value>]
Configures the settings for the specified embedded block RAM component and memory initialization file.
Configure DRAM
eco_config memdram -instance <instance name> -mem <file name>
-format <hex|bin|add> -depth <depth value> -widtha <widthA value>
Configures the settings for the specified dynamic RAM component and memory initialization file.
