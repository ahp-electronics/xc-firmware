Selecting Objects

There are multiple ways to select an object in EPIC. In order to perform actions on them, however, you must select them in the editing area of the main view.

Using your left mouse button, click the desired object in the editing area. To select a single wire on a net, use the Alt+click combination.
In the Object menu, select a category from the drop-down menu, and then select the desired object name. To enlarge the object in the editing area, click the Zoom to button. To perform an action on the object, click on it in the editing area. See Selecting Objects with the Object Menu.
In the Tcl console, select the hyperlink for an object that has been previously selected. See Re-selecting Objects in the Tcl Console
Use the Find dialog box to search for and simultaneously select an object. To perform an action on the found object, click on it in the editing area. See Searching for Objects.

Unplaced components are not displayed in the main view and can only be selected from the Object menu. The types of objects that can be selected include sites, site pins, components, component pins, nets, paths, buses, macros, switch box pins, wires (pin wires, long lines, local lines), and route lines.

When you select an object, it changes color to indicate selection. When selected in the main view, the object's type and name is displayed in the Tcl console, and it is hyperlinked so that you can quickly return to it later. If the object is a used wire (pin wire, long line or local line), the name of the net using the wire is also displayed. When you select a used component pin (or net pin), the Tcl console shows the pin name, the name of the net to which this pin belongs, and the delay time from the net source to this pin.

You can select multiple objects using the Ctrl+Click combination or the Wildcard Filter option in the Object menu. The objects you select must be appropriate for the action you are performing.
