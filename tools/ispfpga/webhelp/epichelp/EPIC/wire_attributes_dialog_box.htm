Wire Attributes Dialog Box

To select a single wire in the editing area, use the Alt+click combination. After a wire is selected, clicking the attribute button on the Actions panel or choosing Design > Attributes will open this dialog box.

Feature
Description
Type
Indicates whether the selected wire is a PINWIRE, LOCALWIRE, or LONGWIRE.
Net Name
If the wire is used, indicates the net routed along the wire. If the wire is unused, NO NET appears in this field.
OK Button
Closes the dialog box.
Help
Opens the online help topic for this dialog.
