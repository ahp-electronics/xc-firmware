Selecting Objects with the Object Menu

The Object menu opens by default at the beginning of your EPIC session. If it is not open, choose View > Object. In this view, objects are grouped into lists of the following types.

Type
Description
All Nets
All nets in the design
Routed Nets
Fully routed nets only
Unrouted Nets
Partially or fully unrouted nets only
All Comps
All components in the design
Placed Comps
Placed components only
Unplaced Comps
Unplaced components only
All Macros
All macros in the design
Placed Macros
Placed macros only
Unplaced Macros
Unplaced macros only
Buses
All buses in the design
Paths
All paths in the design
Preferences
Associated preferences in the design
To select objects using the Object menu:
1. Select an object category, such as Placed Comps, from the drop-down menu.
2. Click the name of an object listed within the selected category.
If you selected a net, it will be highlighted in red in the main window. If needed, click the Zoom Fit button on the toolbar to display the complete layout or click the Zoom To button to center the complete net in the layout.
If you selected a comp, click the Zoom To button so that it is visible in the main window.
Use the Wildcard Filter option at the bottom of the dialog if you are searching for a particular element for which you know the name. For example, enter d* and press the Enter key to filter the list to just elements that begin with the letter “d”.
3. To perform an action on the object, or to cause it to be recorded in the Tcl console with a hyperlink, click on the object in the editing area.
