Pin Attributes Dialog Box

After a pin is selected, clicking the attribute button on the Actions panel or choosing Design > Attributes will open this dialog box.

Feature
Description
Name
Pin Name
Site Name
Indicates the name of the site where the pin resides. The name typically appears in a row/column format; for example, R36C4 in a site name would indicate that it is located in row #36 and column #4.
Component Name
Identifies the name of the component on which the pin resides.
Type
Specifies the function of the pin within the macro.
Net Name
Indicates the name of the net to which this pin belongs.
OK Button
Closes the dialog box.
Help
Opens the online Help topic for this dialog box.
