Save As Dialog Box

The Save As dialog box appears after you select the menu item File > Save As, or press Ctrl+Shift+S.

Feature
Description
Design File Browse Button
Allows you to navigate to a directory location for the physical design (.ncd) file.
Preference File Browse Button
Allows you to navigate to a directory location for the physical preference (.prf).
OK Button
Saves the files and closes the dialog box.
Cancel Button
Ignores any changes you specified and closes the dialog box.
Help
Opens the online help topic for this dialog box.
