Manual Routing

Manual routing allows you to specify the path the signal routing will take. You select the net pins or pin wires to be connected and the routing resources to be used (for example, long lines, local lines), and the system routes the path you have specified. As part of the manual routing, you can select unused pins on placed components, and route these pins together to form a new net, or route them to an existing net.

To route manually:
1. Select the net pins, pin wires, local lines, long lines, route segments, and unused component pins you want to route.
The objects must be selected in the order in which you want them routed. Keep in mind that connections are routed one-by-one from each selected object to the next selected object; for example, if you select three objects in the order A, B, C, you are indicating that you want to route A to B, and then route B to C.
2. Choose Design > Route > Manual Route or click the route button on the Actions panel.
The selected objects are routed in the specified order.
