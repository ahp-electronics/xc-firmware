Editing PFU Logic

A logic block view of a selected PFU slice enables you to edit connections within the slice as well as LUT equations. To make such changes, select a PFU slice component and open the Logic Block view.

Editing Connections

To edit the connections within the PFU slice, you select the component pins within the schematic logic block view. Different pins have different symbol shapes. You can select any input pin, which is symbolized by a triangle or a square. If a group of pins are mutually exclusive, only one can be selected. If you select a pin, the symbol shape will be filled to show the selected state. The following table lists the pin types and the required editing operation.

Changing Logic Equations

The Equations Dialog Box in the Logic Block view enables you to view and edit LUT equations.

To change LUT Equations:
1. In the Logic Block view, click the Equation button on the toolbar.
The Equations dialog box appears, showing the logic equations for the LUT. All possible strings are shown, even though for some modes certain strings have no meaning. The valid operators are:
* represents Logical AND.
+ represents Logical OR.
@ represents Logical XOR.
~ represents Logical NOT.
2. In the text boxes for the LUTs, edit the equations and click OK.
Note
If you make entries to any of the fields in this dialog box, they must be in upper case letters. For example A0+A1 is legal; a0+a1 is not.
Checking Edits

After changing the PFU logic, click the DRC button on the Logic Block view toolbar to run design rule check. DRC results appear in the Tcl console.
