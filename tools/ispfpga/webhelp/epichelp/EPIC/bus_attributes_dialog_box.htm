Bus Attributes Dialog Box

After a bus is selected, clicking the attribute button on the Actions panel or choosing Design > Attributes will open this dialog box. The Bus Attributes dialog box allows you to modify settings and preferences of selected bus.

Feature
Description
Name
The name of the bus.
Elements
The names of the components and nets included in the bus.
Lock Bus
Applies a LOCK BUS preference to the bus, which prevents it from being unrouted, moved, or deleted.
Block
Applies a BLOCK BUS preference, which prevents timing analysis on the selected bus, all nets in the bus, and all paths through the nets defined by the bus.
Use Longline
Applies a USE LONGLINE preference to the bus, which will route each net in the bus along a long line.
Prioritize
Applies a PRIORITIZE BUS preference, which assigns a weighted importance (0-100) to the bus. The default is 3. A priority of 3 or less is not considered critical.
Maxdelay
Applies a MAXDELAY preference to the bus. The MAXDELAY preference specifies a maximum total delay for the driver-to-load connections on all of the nets in the bus.
Maxskew
Applies a MAXSKEW preference to the bus. The MAXSKEW preference specifies a maximum signal skew between a driver and the driver’s loads on a specified clock signal. When applied to a bus, it sets the maximum clock skew for all of the nets in the bus.
OK Button
Applies the given options to the current design and closes the dialog box.
Cancel Button
Ignores any options you may have specified and closes the dialog box.
Apply Button
Applies the selected options to the current design without closing the dialog box.
Help
Opens the online Help topic for this dialog.
