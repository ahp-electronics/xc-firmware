Creating New Designs or Macros

When you create a new design or macro in EPIC, you select the targeted device, package, and performance grade for the design and specify the required file names.

To create a new design:
1. Choose File > New > New Design.
The Create New Design dialog box appears.
2. Enter the design name in the Design file text box. If you do not enter the .ncd extension, it is automatically added when the file is created. To place the new design in a directory other than the current working directory, use the Browse button to select your target directory and specify the file name.
3. Enter the preference file name in the Preference file text box, or use the browse button to specify the directory and file name. If you do not enter the .prf extension, it is automatically added to your file name.
4. Select the desired architecture, device, package, and performance grade from the lists and click OK.
To create a new macro:
1. Choose File > New Macro.
The Create New Macro dialog box appears.
2. Enter the macro name in the Macro file text box. If you do not enter the .nmc extension, it is automatically added when the file is created.
3. To place the new macro in a directory other than the current working directory, use the Browse button to select your target directory and specify the file name.
4. Select the desired architecture, device, package, and performance grade from the lists and click OK.
