Searching for Objects

The Find dialog box enables you to locate and select objects on the schematic layout.

To find objects in the design:
1. Choose Edit > Find or click the find button on the Actions panel. The Find dialog box appears.
2. Type an object name in the Find What text box. You can use wild cards with the object name.
3. Select an object type from the Find Type drop-down list. If you select Site, select the Site Type from the drop-down list that appears.
4. Select Match Whole Word and Match Case as desired, to narrow your search to a complete name or to match upper and lower case letters.
5. Click Find Next to locate the first instance of the object.
The object is highlighted in red in the main view.
6. Use the Find Next or Find Pre buttons to locate instances in the main view. To highlight all instances click Select All.
When you click Close, the main view still shows the last found instance highlighted in red.
If you have located an object you want to edit or perhaps come back to later, be sure to click on the object in the editing area. This will make it available for editing and record its location in the Tcl console with a hyperlink.
