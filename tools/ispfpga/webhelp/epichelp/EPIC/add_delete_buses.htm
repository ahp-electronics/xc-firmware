Adding and Deleting Buses

A bus is a group of nets. You can define buses within EPIC for timing analysis and timing-driven operations. When you add a bus in EPIC, you define a bus using the DEFINE BUS preference. When you delete a bus, you remove the DEFINE BUS preference that defines the bus. You also remove all preferences that directly depend on the DEFINE BUS preference, such as PRIORITIZE BUS.

To add a bus:
1. Select the nets that will be included in the bus.
2. Choose Edit > Add Bus
To delete a bus:
1. In the Object menu, select the bus you want to delete.
2. Click the delete button on the Actions panel or choose Edit > Delete. The bus is deleted.
The next time you save the design, the DEFINE BUS preference that originally defined the path is commented out in the preference file.
