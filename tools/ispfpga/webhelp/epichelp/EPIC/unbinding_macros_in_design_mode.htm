Unbinding Macros in Design Mode

When you unbind a macro, you disassociate the macro's components and nets from the macro. The macro name disappears from the design's database. After unbinding, all of the components and nets formerly in the macro are treated as separate components and nets. Also, component names and net names are changed, replacing the forward slash (/) that indicates the macro's library file with an underscore (_).

To unbind a macro:
1. In the Objects dialog box, display a list of macro names.
2. Click on the macro list items to unbind.
3. Choose Edit > Unbind. For each of the selected macros, the included components and nets are no longer associated with the macro, and the macro name disappears from the Objects dialog box.
