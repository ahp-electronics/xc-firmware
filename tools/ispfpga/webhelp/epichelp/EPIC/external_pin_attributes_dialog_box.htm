External Pin Attributes Dialog Box

The External Pin Attributes dialog box can be called up in both Macro Mode and Design Mode (see Design Modes), but behaves differently.

Macro Mode

In Macro Mode, adding an external pin opens this dialog box. After an external pin is defined, the dialog box can be accessed by selecting the external pin and then clicking the attribute button on the Actions panel or choosing Design > Attribute.

Feature
Description
Name
Pin name
External Name
Determines the name by which this pin will be known when the macro library file is instantiated in a design.
Component Name
Indicates the name of the component containing the pin.
Site Name
Site name of external pin. Indicates the site in which the component containing the pin is placed.
Type
Type of pin function. Specifies the function of the pin within the macro. See Pin Types.
Apply Button
Applies the given options to the current design and allowing you to perform other options or commands before leaving the dialog box.
OK Button
Applies the given options to the current design and closes the dialog box.
Cancel Button
Ignores any options you may have specified and closes the dialog box.
Help
Opens the online help topic for this dialog.
Design Mode

In Design Mode, after a macro is inserted, selecting its external pin and clicking the attribute button on the Actions panel or choosing Design > Attributes opens this dialog box. The dialog options are the same with those in Macro Mode, but all the options are read-only.
