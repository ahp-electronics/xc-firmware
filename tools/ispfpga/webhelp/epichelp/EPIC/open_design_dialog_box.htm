Open Design Dialog Box

The Open Design dialog box appears after you choose Open Design button in the EPIC Start dialog box and select a design. It allows you to select the design file and its associated preference file.

Feature
Description
Design File Browse Button
Allows you to navigate to a physical design (.ncd) file.
Preference File Browse Button
Allows you to navigate to a physical preference (.prf) file.
Auto Lookup
When enabled and no preference file is specified, EPIC assumes that the preference file has the same filename (but different extension) as the design file.
OK Button
Applies the given options to the current design and closes the dialog box.
Cancel Button
Ignores any options you may have specified and closes the dialog box.
Help
Opens the online Help topic for this dialog box.
