Exporting the Main View Image

EPIC allows you to capture the schematic image in the main view at any time and export it as a graphics file such as a bitmap (.bmp) or portable network graphics file (.png).

To export the main view schematic image:
1. Click the Export button on the EPIC toolbar.
The Export Image File dialog box displays the image file types that are available for exporting, in the “Save as type” box.
2. Type a file name and appropriate image file extension in the File name text box, and then click Save.
