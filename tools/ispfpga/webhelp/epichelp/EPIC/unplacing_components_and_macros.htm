Unplacing Components and Macros

The Unplace command removes selected components and macros from their current sites. You can also unplace all placed components and macros in the design. The list of unplaced components can be viewed in the Objects menu. Before the software unplaces a component, it unroutes each component pin on the component.

To unplace selected components:
1. Select the desired components from the Object menu or in the editing area of the main view.
2. Choose Design > Place > Unplace.
The selected components are unplaced, and the component names are displayed in the list of Unplaced Comps in the Object menu.
Note that locked components cannot be unplaced.
To unplace all placed components and macros in the design:
Choose Design > Place > Unplace All.
All placed components and macros in the design except locked ones are unplaced.
