Placement

Components and macros can be placed in the editing area of the main view either automatically or manually. When you use manual placement, you select both a source component or macro and the destination site in which to place the component or macro. When you use automatic placement (Autoplace All), the system selects appropriate sites for all remaining unplaced components in the design. The automatic placement software selects sites intelligently based on routability.

You can also unplace components and macros. The Unplace command unroutes each net pin on the components or macros you select, and then it removes the components and macros from the editing area and adds them to the list of the design's unplaced components or macros.

By default, automatic placement is enabled in EPIC.
