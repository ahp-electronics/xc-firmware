Main Window

The EPIC main window provides a graphical display of the FPGA device, plus tools for selecting, viewing, and editing objects. Descriptions of each section of the main window are provided below the following image.

 
Editing Area

The editing area is the large graphical layout view of the FPGA device that enables you to edit the design. It shows all of the components and interconnections in the design, as well as sites. Items that you select in the editing area or from the Object menu are highlighted on the layout. When you hold your mouse over a component, site or net, a tool tip is displayed that shows the object name.

Note
In the editing area, some labels next to component graphics are shifted downward relative to the component positions. The shift is more apparent at higher magnification.
Object Menu

The Object menu lists by category all nets, comps, macros, buses, paths, and preferences in the design and allows you to select individual objects and highlight them in the editing area. After selecting an object from the Object menu, you can magnify it in the editing area by clicking the Zoom To button on the Zoom toolbar.

Actions Panel

The Actions panel enables you to perform many actions for selected objects, such as adding and deleting, routing and unrouting, highlighting, viewing attributes or pin pair delay, and performing design rule check.

Tcl Console Tab

The Tcl console tab at the bottom keeps track of all actions. The Tcl console lists with a hyperlink each component or net you have selected in the editing area, enabling you to immediately return to previous selections. The Tcl command prompt enables you to perform many actions in EPIC. All commands that you enter are displayed in the Tcl console and automatically saved. See EPIC Tcl Commands.

Info, Warning, and Error Tab

See Viewing Info, Warning and Error Messages.

World View

The World View is a miniaturized global view of the device. It highlights the items that have been selected and shows where they are positioned within the device layout. After you have zoomed in to a location in the editing area, you can pan across the layout by dragging the inner locater box within the World View.

Zoom Toolbar

The Zoom toolbar enables you to control the display of the editing area of the main view by zooming in and out, zooming in to a selected object, and returning to the full layout view. See Zooming in the Main View.

Layers Toolbar

The Layers toolbar gives you control over the types of objects that are displayed on the editing area layout. By default, all sites, components, nets, and ratsnests are displayed in the editing area. You can also display local wires, long wires, pin wires, switch boxes, and switches by selecting these items from the Layers toolbar. See Object Definitions.
