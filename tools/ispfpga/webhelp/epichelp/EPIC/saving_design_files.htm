Saving Design Files

At any time during an EPIC session, you can save the design or macro on which you are working. You can also save your physical design (.ncd) file as a macro (.nmc) file.

To save changes to the design or macro:
Choose File > Save.
To save a design using a different name or location:
1. Choose File > Save As or press Ctrl+Shift+S.
2. In the Design File text box, type a new name for the .ncd file if you’re saving a copy of it in the current location. If desired, click the Browse button and navigate to a different directory for the .ncd file.
3. Enter a preference file name and, if necessary, the full path. If you do not enter the full path, the file is saved in the current working directory.
4. Click OK.
To save a macro using a different name or location:
1. Choose File > Save As.
2. In the Save As Macro dialog box, type a new name for the .nmc file if you’re saving a copy of it in the current location. Click the Browse button to save the macro in a different location.
3. Click OK.
To save your design as a macro library file:
1. Choose File > Save As Macro and type a name for the .nmc file.
2. Click the Browse button to save the macro in a different location.
3. Click OK.
To be able to use this macro when you open it, you must assign one or more external pins. See Adding External Pins in Macro Mode. Preferences are not retained when you save a design file as a macro library file. No preference file is generated, even if the design had preferences defined.
