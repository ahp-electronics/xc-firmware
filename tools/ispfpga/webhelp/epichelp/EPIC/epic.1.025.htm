Viewing Info, Warning and Error Messages

EPIC generates log files for all project activities similar with that in the Diamond output panel. The log files contain processing information, as well as error and warning messages.

Info, Warning and Error Messages

The following actions are provided in the Info, Warning and Error windows.

Copy: copy selected content onto clipboard.
Select All: select all messages in current window.
Clear All: clear all messages in current window.
Find in Text: show a find bar for user to find special messages.
Help: provide message on-line function.
