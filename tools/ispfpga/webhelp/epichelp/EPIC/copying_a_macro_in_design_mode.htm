Cloning a Macro in Design Mode

When you clone a macro in design mode, you add a duplicate of the macro instance to the design. The source is a placed or unplaced macro. The target is a vacant site in which to place the reference component of the copied macro.

To clone a macro in design mode:
1. In the Object Menu, select the name of the macro that you want to clone.
2. Select the vacant site into which you want to copy the source macro's reference component. This is the target.
3. Choose Edit > Clone.
The reference component of the source macro is copied to the selected site, and the other components in the macro are copied in positions relative to the reference component.
If you do not specify a name for the copy, the name assigned to the new macro is in the format $libfile_number, where libfile is the name of the macro library file of the source macro and number is a number assigned to each new copy of the source. Numbering starts at 0, and increases by one for each new copy of the macro. For example, if you make multiple copies of a macro called “add,” which is an instantiation of macro library file “adder,” the first copy will be named $adder_0 and second will be named $adder_1, and so on.
