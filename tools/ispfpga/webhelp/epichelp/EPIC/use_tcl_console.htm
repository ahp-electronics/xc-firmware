Using the Tcl Console Tab

The EPIC Tcl console keeps track of all actions executed in the current session, as well as Info, Warning and Error messages, and displays all of them. When you select an object, such as a net, wire, or component, it records these selections and hyperlinks them so that you can quickly return to them by clicking the hyperlinked name in the console window.

Automatic Recording of Commands

The Tcl console also displays all of the commands that you perform on a design, such as adding an object, moving or swapping components, or performing DRC. All of the commands for an EPIC session are automatically saved into a .tcr file in the project directory. If a loss of power or a software error causes the EPIC session to exit abnormally, all the commands that were used in the session will still be available in the .tcr file.

The .tcr files are located in the following directories:

..\<project_name>.ncd_tcr.dir

..\<project_name>.nmc_tcr.dir
