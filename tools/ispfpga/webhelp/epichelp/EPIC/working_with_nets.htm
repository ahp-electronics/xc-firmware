Adding and Deleting Nets

EPIC enables you to add nets to a design or macro and add component pins to an existing net. You can also delete one or more nets from the design database.

Adding Nets

When adding nets, each of the selected pins must be on a placed component, and the selected pins must not belong to any other nets.

To add a net to the design:
1. Select all the component pins that will be included in the new net.
2. Click the add button on the Actions panel or choose Edit > Add.
The selected pins are joined as a net, and a confirmation, along with the name of the new net, is shown in the Tcl console; for example, net: $SIGNAL_4.
The new net is routed as completely as possible.

When you add nets, the name assigned to the first new net is $SIGNAL_0; subsequently added nets will be named $SIGNAL_1, $SIGNAL_2, etc.

Deleting Nets

When you delete a net, all net pins on the net are unrouted, and then each net pin is removed from the design database. Finally, the net itself is removed from the database. You can also delete multiple nets.

To delete a net:
1. In the Object menu, select the nets to delete. Alternatively, select the nets in the editing area of the main view. Use the Ctrl+click combination to select multiple nets.
If you are not sure of the name of a net you want to delete, select a pin, ratsnest line or route on the desired net in the editing area. The signal name appears in the Tcl console.
2. Click the delete button on the Actions panel or choose Edit > Delete.
The nets are eliminated from the database.
