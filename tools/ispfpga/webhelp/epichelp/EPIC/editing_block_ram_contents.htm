Editing Block RAM Contents

An EBR (embedded block RAM) element is a RAM cell in an FPGA that can be configured for RAM, ROM, FIFO, CAMs, and so forth.

When you open the Logic Block view for EBR elements, a tabular view is displayed, which allows you to change attributes such as DATA_WIDTH (Data Word Width), REGMODE (Register Mode for Pipelining), RESETMODE (Reset Type), or CSDECODE values of EBR elements. You can also enable/disable the global set/reset (GSR) and change memory initialization values (INITVAL). The content of the memory initialization values is the same format as an INITVAL constraint in a .prf file.

See the FPGA Libraries Manual for details on attribute settings for individual EBR elements. For details about EBR port definition, attribute definition and their usage, refer to the following Technical Notes:

LatticeECP2/M Memory Usage Guide (TN1104)
LatticeECP3 Memory Usage Guide (TN1179)
On-Chip Memory Usage Guide for LatticeSC Devices (TN1094)
Memory Usage Guide for LatticeECP/EC and LatticeXP Devices (TN1051)
Memory Usage Guide for MachXO Devices (TN1092)
Memory Usage Guide for MachXO2 Devices (TN1201)

The memory ports can be set to SINGAL, INVERT, POWER, or GROUND.

You should do a DRC check after making changes. Click the DRC button on the Logic Block view toolbar. DRC results appear in the Tcl console.
