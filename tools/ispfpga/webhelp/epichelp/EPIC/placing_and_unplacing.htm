Placing and Unplacing

Components and macros can be placed in the editing area of the main view automatically or manually. The placement commands are available from the Design > Place sub-menus. When you use manual placement, you select both a source component or macro and the destination site in which to place the component or macro. When you use automatic placement (Auto Place All), the software selects appropriate sites for all remaining unplaced components in the design. The automatic placement software selects sites intelligently based on routing capability.

You can also unplace components and macros. The Unplace command unroutes each net pin on the components or macros you selected, and then it removes the components and macros from the main view and adds them to the list of the design's unplaced components or macros.
