Automatic Placement

By default, automatic placement is enabled in EPIC for all unplaced components. To place only selected unplaced components, you will need to use manual placement.

To automatically place all unplaced components:
Choose Design > Place > Autoplace All.
All of the design's unplaced components and macros are automatically placed in vacant sites in the editing area.
Note
When you autoplace all remaining components within EPIC, no existing routing is ripped up and rerouted to accommodate the new placement. Since PAR rips up routes if necessary, placement done by running PAR will produce different results than placement done in EPIC. PAR places components more efficiently, and a densely packed design can more easily be placed and routed in PAR than in EPIC.
