Setting Preferences in the User Interface

Certain net and component preferences can be set using the appropriate Attribute dialog box in the EPIC user interface.

The following table lists these preferences and the corresponding Attribute dialog boxes.

Preference
Accessed Through
BLOCK BUS
Bus Attributes dialog box.
BLOCK NET
Net Attributes dialog box.
BLOCK PATH
Path Attributes dialog box.
DEFINE BUS
Created, edited, and removed with add and delete; viewed with Bus Attributes dialog box.
DEFINE PATH
Created, edited, and removed with add and delete; viewed with Path Attributes dialog box.
LOCATE COMP
Component Attributes dialog box.
LOCATE MACRO
Macro Attributes dialog box.
LOCK BUS
Bus Attributes dialog box.
LOCK COMP
Component Attributes dialog box.
LOCK MACRO
Macro Attributes dialog box.
LOCK NET
Net Attributes dialog box.
MAXDELAY BUS
Bus Attributes dialog box.
MAXDELAY PATH
Path Attributes dialog box.
MAXSKEW BUS
Bus Attributes dialog box.
PRIORITIZE BUS
Bus Attributes dialog box.
PROHIBIT SITE
Site Attributes dialog box.
USE LONGLINE BUS
Bus Attributes dialog box.
USE LONGLINE NET
Net Attributes dialog box.
