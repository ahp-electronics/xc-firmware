Equations Dialog Box

The Equations dialog box opens when you click the Equations button in the Logic Block view. The valid operators are:

* represents Logical AND.
+ represents Logical OR.
@ represents Logical XOR.
~ represents Logical NOT.

The Logic Equations dialog box is on the left. If the element is a block RAM element, a dialog box like the one on the right will appear after clicking the Properties button in the Logic Block view.

Feature
Description
Equations
Contains editable logic equations. These must be edited using all capital letters.
Address
Contains editable memory location address information.
Memory info
Shows uneditable logic program and size information.
OK Button
Applies the given options and closes the dialog box.
