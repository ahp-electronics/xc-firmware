Routing

Connections can be routed in the editing area of the main view automatically or manually. By default, automatic routing is turned on. With automatic routing, the software selects the routing path for additions or changes to nets, components, or macros. When you use manual routing, you specify the routing path to be used.

You can also unroute connections. Unrouting rips up any specified routes without eliminating the logical connections in the nets involved. You can also route or unroute all connections in the design.
