Getting Started

The EPIC Device Editor is a stand-alone tool. You launch EPIC by running the EPIC executable or its shortcuts. Using the Start menu on a PC, you can open EPIC from the Accessories folder of the Diamond installation. For Linux, use the command line to change to the desired directory and open EPIC, as shown in the following example:

cd ../diamond/3.1_x64/bin/lin64/
./diamond -epic

When you start a new EPIC session, the EPIC main window appears with no design loaded. Use the File menu to open a design or macro or to create a new design or macro. Alternatively, use the Tcl command prompt beneath the Tcl console to load a design or macro or create a new one. See the EPIC Tcl Commands section for syntax and descriptions of all the Tcl commands that are available with EPIC.
