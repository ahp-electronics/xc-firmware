Routing and Unrouting Macros in Design Mode

Routing and unrouting macros is similar to routing and unrouting components. See Routing and Unrouting.

Some notes about routing and unrouting macros in design mode:

To select a macro for automatic routing, select it from the Placed Macros section of the Object menu.
When you place a macro, any pre-routed connections that exist in the macro's library file are automatically routed after the macro is placed. You cannot unroute these connections in design mode.
When you automatically route a macro, each external pin connected to a net is routed. Also, any nets that are not pre-routed are automatically routed.
