Placement and Routing in Macro Mode

In macro mode, you can place components in the editing area of the main view and route the connecting nets. When you later instantiate the macro library file in a design, the placement of the components relative to each other will be maintained in the instantiated macro. Also, routing in the instantiated macro will follow the routing specified in the library file.
