Defining Reference Components (Macro Mode)

To define a reference component:
1. Select the component that you wish to be the reference component.
2. Choose Design > Macro > Set RefComp to indicate that the selected component is to be the reference component.
The selected component is now the reference component.
Note
You can change the reference component at any time while EPIC is in macro mode. If you do not designate a reference component, the component in the lower left corner will be selected as reference component when the macro file is saved.
