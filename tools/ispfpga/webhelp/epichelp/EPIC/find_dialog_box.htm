Find Dialog Box

The Find dialog box can be accessed by choosing Edit > Find or pressing Ctrl+F. It allows you to search for an object of a specific type.

Feature
Description
Find What
Enter the name of the object you want to find.
Find Type
Select an object type.
Site Type
Select a site type. This menu is displayed when Site is selected in the Find Type drop-down menu.
Match Whole Word
Narrows the search to a complete name match.
Match Case
Narrows the search to words that match upper and lower cases.
Find Next
Finds the first or next instance of the object and highlights it on the editing area layout.
Find Pre
Searches backward to find the previous instance and highlights it on the editing area layout.
Select All
Selects all objects of the specified type and name and highlights all of them on the editing area layout.
Close Button
Closes the dialog box and keeps the found objects highlighted on the editing area layout.
