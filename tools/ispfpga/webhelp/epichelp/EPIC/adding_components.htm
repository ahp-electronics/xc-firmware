Adding and Deleting Components

EPIC enables you to add components to your design and delete them. When you add a component, it becomes a part of the design database.

When you delete a component, each net pin on the component is unrouted, after which each net pin is removed from its net. Finally, the component is removed from the design database.

To add a component to a design:
1. In the editing area, select a vacant site in which to place the new component.
2. Click the add button on the Actions panel or choose Edit > Add.
3. In the Add Component dialog box, type a unique name for the new component. The component name box must not be empty.
4. Click OK.
To delete one or more components:
1. Select the components to delete.
2. Click the delete button on the Actions panel or choose Edit > Delete. The components are eliminated from the database.
Note
It is not possible to delete a component that is part of an instantiated macro in design mode.
