Routing and Unrouting

Connections can be routed in the editing area of the main view automatically or manually. When you select automatic routing (Auto Route), the software selects the routing path for each connection you select. By default, automatic routing is turned on. With automatic routing, objects are automatically routed when you add, place, copy, swap or clone. When you use manual routing, you specify the routing path to be used.

You can also unroute connections. Unrouting rips up any specified routes without eliminating the logical connections in the nets involved. You can also route or unroute all connections in the design, by selecting Auto Route All or Unroute All.
