Moving Components

In order for a component to be moved, it must be a placed component, and it must be moved to a vacant site. The component and the site must be of the same block type.

To move a component to a vacant site:
1. Select the component that you want to move.
2. While holding the Ctrl key, select the vacant site to which you wish to move the component.
3. Choose Edit > Swap or click the swap button on the Actions panel.
The component is moved to the unused site and routed.
