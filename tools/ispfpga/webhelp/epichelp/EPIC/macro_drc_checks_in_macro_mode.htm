Macro DRC Checks in Macro Mode

There are a number of physical DRC checks that test the macro you are editing. You can execute these tests during your EPIC session. The tests will run automatically whenever you save the macro library file.

To run the macro DRC check:
1. Make sure nothing is selected in the editing area or the Objects dialog box.
2. Choose Design > DRC > DRC or click the drc button on the Actions panel.
EPIC runs the macro DRC, along with other DRC tests. The DRC results are displayed in the Tcl console.
