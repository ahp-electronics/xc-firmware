Macro Attributes Dialog Box

In Design Mode, after you have selected a macro, clicking the attribute button on the Actions panel or choosing Design > Attributes will open this dialog box. Adding a macro will also bring up this dialog box.

The Macro Attributes dialog box allows you to modify settings and preference of selected macro. Macro Attributes are explained in first table. Macro Preferences options are listed in the second table.

The preferences section of this dialog box is hidden by default. It can be toggled on by clicking the Show Preferences Button, which will be replaced by a corresponding Hide Preferences button when this level of detail is displayed.

Macro Attributes

Feature
Description
Macro Name
Specifies a name for the macro instance. If you do not enter a name, a name will be assigned automatically when the macro is added to the design. If you change the macro name, all of the names of the components, nets, and external pins that are part of the macro will be updated automatically to reflect this name change.
Macro Library File
Also, generally referred to as “macros” or “physical macro files”. Macro library file is a file containing the definition of a macro. Macro library files, which have an .nmc extension, are created and edited in EPIC, operating in macro mode.
Status
Indicates whether the macro is UNPLACED or PLACED. A macro is only shown as PLACED if all the components in the macro are placed.
Reference Comp
Indicates the component designated as reference component. A macro with no pre-placed components will not have a reference component, and a macro with at least one pre-placed component will have a reference component. The reference component is described in the Macro Terminology topic. This field holds up to 50 characters, is non-modifiable, and scrollable.
External Pins
External Pins are macro pins used to connect the components in an instantiated macro to other components in the design (outside of the macro).
Show Preferences
Displays all the preferences that applied to the type of object you are editing or adding.
Apply Button
Applies the given options to the current design and allowing you to perform other options or commands before leaving the dialog box.
OK Button
Applies the given options to the current design and closes the dialog box.
Cancel Button
Ignores any options you may have specified and closes the dialog box.
Help
Opens the online help topic for this dialog box.

Macro Preferences

Feature
Description
Lock
If enabled, indicates a lock or global lock placement preference that is applied to the applicable object (i.e., site, comp, or macro). A locked component cannot be unplaced, unrouted, moved, swapped, or deleted. If a LOCK PLACEMENT preference currently applies to the design, the Lock toggle button is enabled, and you cannot unlock the component using the toggle button. If a macro is locked, all macro components that were pre-placed (that is, components that were placed in the macro's library file) are locked. The macro cannot be unplaced.
Locate
If enabled, indicates a locate preference that is applied to the applicable object (i.e., site, comp, or macro).The LOCATE preference places a component at a specified site and locks the component to the site (see Lock preference).A locate macro preference places the macro's reference component at a specified site, places all of the macro's pre-placed components (that is, all components that were placed in the macro's library file) in sites relative to the reference component, and locks all of these placed components at their sites.
Paths
Indicates any defined paths (i.e., paths defined by define path preferences) that include this comp or macro.
